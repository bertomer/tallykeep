# Future iterations — retired

Per ADR-0014, the flat `future_iterations.md` file is retired in
favor of the `backlog/` folder (one file per captured idea).

See `backlog/README.md` for the new conventions, lifecycle, and
iteration roadmap.

This file is kept temporarily so cross-references from canonical docs
(e.g. `next_iteration.md`'s scope-out section) still resolve until a
follow-up spec-agent session cleans them up. The file is deletable
after that pass.
