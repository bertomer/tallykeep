Connect a provider

An exchange, broker, or custodial bank where you hold funds. TallyKeep observes your balance with a read-only key. Automated withdrawal can be configured separately.

Create a read-only API key on Kraken Pro

Settings → Connection and API → Create an API key. Name it TallyKeep Read and tick only Query funds.

This key has too many permissions

We accept only read access to keep your account exposure to a minimum. This one also has:

  • Withdraw funds
  • Trade

Replace these keys on Kraken Pro with ones that have ONLY Query funds ticked. Then paste the new pair here.
