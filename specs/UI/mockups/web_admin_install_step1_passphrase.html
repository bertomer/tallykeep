What this passphrase does
  • Encrypts your server's secrets. Custodial-provider API keys and anything else sensitive on disk. The server cannot start without you typing this.
  • Unlocks your phone if biometric fails. Your phone asks the server to validate; the passphrase itself never leaves the server's memory.
