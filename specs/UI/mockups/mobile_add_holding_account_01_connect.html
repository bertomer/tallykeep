Connect a provider

An exchange, broker, or custodial bank where you hold funds. TallyKeep observes your balance with a read-only key. Automated withdrawal can be configured separately.

Create a read-only API key on Kraken Pro

Settings → Connection and API → Create an API key. Name it TallyKeep Read and tick only Query funds and Query ledger entries.

Copy both keys before closing Kraken's dialog. If you lose one, you'll need to delete this key on Kraken and create a new one.
