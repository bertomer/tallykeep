1 · The mark

  • Form: medieval split tally stick — stock + foil, arranged as a Y (the "Y" of TallY)
  • Wood: Aged Oak #A88554
  • Texture: 2 grain stripes per stick (canonical) · solid (favicon ≤16px)
  • Background: Cream #F4EAD5
  • Container: rounded square, radius 14% of width (28/200)
  • Tip cuts: perpendicular straight (3 of 4) · biais at handle (signature)

3 · Color tokens

Aged Oak
#A88554
Wood. Both stock and foil. Single-tone fill.
Grain
#6F5638
Stripe lines along each stick. Stroke 1.4, opacity 0.6.
Cream
#F4EAD5
Background. Also fills the carvings (notches and hanging hole) — they read as cut-throughs to the surface behind.

Rejected colors

  • #B07645 — Saddle Brown. Generic, lives in crowded "artisan / coffee / bourbon" zone.
  • #BD9E60 — Aged Honey. Was the lean before user feedback; too light, didn't read as wood with history.
  • #8B6E45 — Weathered Oak. Too gothic / monastery archive; lost the "in-pocket precision instrument" register.
  • #C9B897 — Whitewashed. Insufficient contrast against cream at 60 px and below.

Rejected texture approaches

  • Turbulence noise filter (feTurbulence) — rasterizes on most SVG-to-PDF exports, browser-inconsistent rendering, breaks in icon-export pipelines.
  • Knots (ellipses) — pedagogically punchier (matches more visually obvious) but starts to read as "illustration of wood" rather than designed mark; competes visually with notches. Archived for potential "how it works" page.

4 · Anatomy

Stock
Longer half. Goes from upper-right (un-handle tip) to lower-left (handle). Historically the creditor's half — the bank kept this.
Foil
Shorter half. Upper-left tip down to broken end. Historically the debtor's piece — a payee carried this. "Got the short end of the stick" = received the foil.
Handle
Rectangular block. Asymmetric, protrudes only on the inner (split-cut) face. The outer face is flush — that's the original outer surface of the unsplit stick. The handle's extra thickness only appears on the side where the cut went through.
Biais cut
Diagonal at the handle tip. The only non-orthogonal cut on the silhouette — it's the visual signature of the mark.
Tips
Stock upper-right, foil upper-left, foil broken end: all perpendicular straight cuts. No rounded line caps anywhere.
Notches
Three V-shaped per stick, on inner edges, at matched positions across the gap. Middle notch slightly wider (medieval denomination convention). They're the tally's actual record-keeping mechanism — the verification metaphor.
Hanging hole
Small circle at the handle's geometric center. Practical detail (sticks hung in archive racks) and adds craft authenticity.

5 · Landing-page interaction

Tap the mark below. Both halves regenerate with new but matching grain. Pedagogical demonstration of the verification metaphor, deployed on the landing-page hero only. Not the canonical icon — the canonical mark uses one locked seed everywhere else.

Tap to verify a new pair
— · —

How it works

Seeded PRNG (xorshift32) generates a deterministic offset array from a seed. Each stripe has {offset, opacity, width}. Both sticks render the same offset on each stripe — each stick computes its own screen-space endpoints in its own local coordinate frame. Same seed → matching grain. New seed → new but still-matching grain. 

// Implementation outline (~80 LOC total)
function makeRng(seed) {
  let s = seed | 0; if (s === 0) s = 1;
  return () => {
    s ^= s << 13; s ^= s >>> 17; s ^= s << 5;
    return ((s >>> 0) / 4294967296);
  };
}

function generateStripes(seed) {
  const rng = makeRng(seed);
  return Array.from({length: 3 + Math.floor(rng() * 2)}, () => ({
    offset:  -8 + rng() * 16,        // perpendicular position
    opacity: 0.45 + rng() * 0.35,
    width:   0.9 + rng() * 1.4,
  }));
}

function stockLine(O) {
  return {
    x1: 163.27 - 0.757 * O, y1: 31.22 - 0.654 * O,
    x2: 42.28  - 0.757 * O, y2: 171.26 - 0.654 * O,
  };
}
function foilLine(O) {
  return {
    x1: 45.73 + 0.756 * O, y1: 31.22 - 0.655 * O,
    x2: 91.28 + 0.756 * O, y2: 83.79 - 0.655 * O,
  };
}

// The clipPath on each <g> ensures lines never extend past the wood.

6 · Decisions log

Form

  • Y arrangement (stock as the long bent piece going from upper-right to lower-left, foil as the upper-left branch).
  • Foil intentionally much shorter than stock — a known simplification of real split tallies (which were near-equal halves), traded for the Y silhouette and the "Y of TallY" reading.
  • Consequence: the verification metaphor only fully applies to the shared length range. v1 grain accepts this quietly; knots variant made it conspicuous (rejected, see §3).

Tip cuts

  • Stock upper-right, foil upper-left, foil broken end: perpendicular straight cuts.
  • Handle tip: biais (diagonal). The only non-orthogonal cut — visual signature.
  • Square tip variant was the alternative; biais won on geometric coherence with the V-notches and "hand-tooled" register.

Handle

  • Rectangular block, not a bulb (early "bulb" handle was scrapped — wrong register).
  • Asymmetric — protrudes on inner face only. v4 had it on the outer face, which was anatomically wrong (the cut removes wood from the inner face, so the unsplit handle naturally extends on that side).
  • Hanging hole at handle's geometric center.

Color

  • Picked: Aged Oak #A88554 — clearly aged, still distinct from cream at 60px, doesn't read as gothic.
  • See §3 for rejected color list and reasoning.

Texture

  • Picked: 2 grain stripes per stick at perpendicular offsets ±5, stroke 1.4, color #6F5638, opacity 0.6.
  • Solid version (no grain) used for favicon ≤16px where grain becomes sub-pixel noise.
  • See §3 for rejected texture approaches.

Notches

  • Three V-shaped per stick, on the inner edge of each.
  • Middle notch slightly wider — medieval denomination convention (different notch widths represented different amounts: pence, shillings, pounds).
  • Three was the chosen number; fewer reads as accidental, more clutters.

Landing-page interaction

  • Tap the mark → both halves regenerate matching grain.
  • Used only on the marketing site hero, not in-app.
  • Caption tested: "The grain matches. A tally stick is split from a single piece of wood. The pattern on both halves is the proof — that's how you knew it was real."
  • Constraint: don't overuse. Once on the landing hero, optionally once again on a "How it works" callback — not as a constant fidget across the site.
