TallyKeep wireframes — v5

Banking-first onboarding. The questionnaire from v4 is gone. Onboarding is now the minimum: passphrase, node, first Holding. Detail/privacy decisions are deferred to contextual moments or to Settings.
Changelog vs v4:
  • Removed Q1 (where is your BTC), Q2 (detail level), Q3 (privacy) from onboarding
  • Onboarding is now: Welcome → Encryption → Node → First Holding (4 steps, not 7)
  • "Add Holding" picker uses banking-flavored descriptions (Vault = safety-deposit box, etc.)
  • Privacy/Blueprint prompt becomes contextual on first Strongbox or Vault creation
  • Detail-level features default to off; one Settings master toggle exposes them
  • "I don't have any Bitcoin yet" path included as an honest stub for v1 ("coming soon" guidance)
See spec_amendments.md for the rationale and the v1-real vs v1-stubbed status table.

Set up your first place to keep money

Pick what you already have. We'll walk you through connecting it. You can add others later.
Account — like an exchange or brokerage
Money you've got at a custodial provider (Kraken, Bitstamp, Coinbase, Swissquote…). We connect via API to read your balance and trigger withdrawals to your own wallets — your funds never leave your control more than they already do.
Purse — like a checking account
An everyday wallet for spending and small amounts, on a phone or laptop. The keys live on your device; we just import the watch-only descriptor.
Strongbox — like a savings account, but you hold the key
A safer spot for medium-term holdings, usually on a hardware wallet (Coldcard, Trezor, Ledger, Jade). We import the watch-only descriptor and prepare PSBTs you sign on the device.
Vault — like a safety-deposit box
A heavily-protected long-term holding. Multi-key, possibly with timelocks or geographic separation. Used rarely, with care.
⚠ v1 stores Vault metadata but only single-key descriptors are accepted. Full multisig support arrives in v2.
I don't have any Bitcoin yet — let's start from zero
We'll point you toward a custodial exchange where you can acquire some, then bring it under TallyKeep. TallyKeep manages Bitcoin you have somewhere — it doesn't broker the buy itself.
